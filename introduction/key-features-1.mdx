# Key Features

The LMS platform provides a rich set of features designed to support **online learning, assessments, and platform administration**.

---

## User & Role Management
- Multiple user roles (Admin, Instructor, Student, etc.)
- Role-based permissions
- User archiving, restoration, and blocking
- Public instructor profiles

---

## Training Management
- Create and manage trainings
- Assign instructors to trainings
- Organize content into chapters and lessons
- Upload images and videos
- Control visibility (public/private)

---

## Exams & Quizzes
- Standalone exams or training-linked quizzes
- Multiple question types
- Automated evaluation
- Exam access control based on purchases

---

## Landing Page Customization
- Fully configurable landing page
- Featured categories and partner companies
- Dynamic ordering of trainings and exams
- Header, banner, and footer customization
- Multi-language support

---

## Orders & Payments
- Cart system for trainings and exams
- Online and offline payment handling
- Coupons and discounts
- Invoice generation (PDF)

---

## Multi-Company Support
- Each company has isolated data
- Company-specific branding
- Platform theme customization per company

---

## Multi-Language Support
- English, French, and Arabic
- Runtime language switching
- Centralized translation keys

---

## API-First Architecture
- Clean REST APIs
- Fully documented endpoints
- Easy frontend-backend integration
